You are creating an Instance entry.

An Instance is an edition or translation of a book. It is the second level of the 3-level model of representing books: Work > Instance > Book. This model enable us to create better linkage between different editions of a book, and build comprehensive bibliography for the author.

An Instance is related to only one Work, but can be related to multiple Books.

Examples

The Annotated Pride and Prejudice, annotated and edited by David M. Shapard is an Instance of Jane Austen's Work.

This Instance has two Books: a hardcover in 2003 and a paperback in 2007.
