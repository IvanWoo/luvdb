Guide to Creating an Instance

Instance: The Edition Level

An Instance represents a specific edition or translation of a Work. It's a unique realization of the abstract Work, capturing details that vary between editions. For information about the physical copy, create a Book entry.

What to Include in an Instance

Include details that define the edition, such as publication date of the instance, language of the instance, etc. Link it to the appropriate Work to build a detailed view of the book's different versions.
