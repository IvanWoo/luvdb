Guide

You are creating a Work entry.

On LÊŒvDB, we use a 3-level model to represent books: Work > Instance > Book. This model enable us to create better linkage between different editions of a book, and build comprehensive bibliography for the author.

A Work is an abstraction that encapsulates the core concept of a book, irrespective of its editions or formats.

A Work entry is not check-in-able. If you want to check in to a book, you need to create a Book entry.

If you're entering specific details about a particular edition, please create an Instance entry.

A Work can be related to multiple instances, and an Instance can be related to multiple books.

Examples

Pride and Prejudice was a Work (a novel) written by Jane Austen, and first published in January, 28, 1813.

David M. Shapard has edited and annotated an Instance of Pride and Prejudice, which published as two books: a hardcover in 2003 and a paperback in 2007. In 2012, a revised and updated Instance of The Annotated Pride and Prejudice was published as a paperback.
