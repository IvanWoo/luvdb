Guide

You are creating a Work entry.

A Work is an abstraction that encapsulates the core concept of a book, irrespective of its editions or formats.

On LÊŒvDB, we use a 3-level model to represent books, similar to BIBFRAME: Work > Instance > Book. This model enable us to create better linkage between different editions of a book, and build comprehensive bibliography for the author.

A Work entry is not check-in-able. If you want to check-in a book, you need to create a Book entry.

If you're entering specific details about a particular edition, please create an Instance entry.

Remember, in our system, only 'Book' entries are check-in-able.

Examples

Pride and Prejudice was a Work written by Jane Austen, and first published in January, 28, 1813.
