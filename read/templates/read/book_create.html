Guide to Creating a Book

Book: The Physical Entity

A Book represents the tangible item, a specific physical copy of an Instance. This is where you log details about the individual volume in hand. If you need to create an Instance first, go to the Instance section.

What to Include in a Book

Log specifics about the physical object, such as covers, publisher, ISBN, etc. Linking it to the corresponding Instance will create a complete record from the conceptual Work to the actual Book.
