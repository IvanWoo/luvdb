Markdown

Markdown is supported in all primary text fields. If you are unfamiliar with markdown, you can find more information about it here.

Special Syntaxes
Spoiler

[text](/s) or [text](/spoiler) will hide the text until clicked on.

Citation

[text](/c) or [text](/citation) will move the text to the left side of the main text. If you want to add hyperlink in the citation, you need to use the <code> tag:

<code>[text](http://example.com)</code>

Caution

There is no auto-save feature available on this site. It is advisable to compose your post in a dedicated offline text editor and then simply paste it here.
