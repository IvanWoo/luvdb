All main text fields support markdown. If you don't know what that is, you can learn more about it here.

Special Syntaxes
Spoiler

[text](/s) or [text](/spoiler) will hide the text until clicked on.

Citation

[text](/c) or [text](/citation) will move the text to the left side of the main text.
