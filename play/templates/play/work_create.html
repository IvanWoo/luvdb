Guide to Creating a Work

Work: An Overview

A Work represents a foundational concept for a game, encompassing all its different releases. If you're aiming to log a specific release, head to the Game section instead.

Consider a Work as an umbrella: various releases, including those on different platforms or remastered versions, fall under it. However, distinctively unique releases should have their separate Game entries, all of which are connected by one Work.

What to Include in a Work

Log the basic details of the game and its creators in a Work. It captures the first release date, regardless of subsequent game completions.
