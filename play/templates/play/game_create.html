Guide to Creating a Game

Game: An Overview

A Game entry represents a specific release or version of a game. If you're aiming to log a higher-level concept that includes all releases of a game, go to the Work section instead.

Think of a Game as an individual instance: one unique release on a platform, or a distinct version like a remastered edition. Multiple Game entries may be linked to a single Work.

What to Include in a Game

Log specific details such as the title, release date, platform, developers, publishers, and any unique attributes like cover art or additional content.
