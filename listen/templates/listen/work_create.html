Guide

You are creating a Work entry.

On LʌvDB, we use a 3-level model to represent a musical work: Work > Track > Release. This model enable us to create better linkage between different releases and recordings of a musical work, and build comprehensive discography for the creators.

A Work is an abstraction that encapsulates the core concept of a music work, it contains only the meta data that is common to different recordings of the work, such as the composers, lyricist, the original performers, etc.

A Work entry is not check-in-able. If you want to check in to a book, you need to create a Release entry.

Examples

The Serenade for Strings, Op. 22 is a Work by Antonín Dvořák. A Work can have multiple tracks, performed by various performers from different time. A track can be the performance of the work in its entirety or just part of it.
